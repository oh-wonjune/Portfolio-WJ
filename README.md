# 포트폴리오 프로젝트 🌟

---

## 프로젝트 개요 📌
포트폴리오 웹사이트 프로젝트, [wonjune.me](http://wonjune.me)
이 프로젝트는 제 개발 역량과 창의성을 보여주는 프로젝트입니다.

---

## 기능 및 특징 🔍
- 개인 포트폴리오를 위한 웹사이트.

---

## 사용 기술 스택 🛠️
이 프로젝트에서 사용된 주요 기술은 다음과 같습니다:
- **프론트엔드**: Next.js, React
- **그래픽스**: OGL
